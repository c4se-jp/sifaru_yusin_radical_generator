# 白色語語根生成器 (sifarU yUsin ru dEku kekurIa)

https://c4se-jp.github.io/sifaru_yusin_radical_generator/

## 辭書

- -ga (接尾辭) -達。幾つかの-
- dEku 根。植物の根。語根
- dEkuga 根達。語根達
- kekurE 作る。造る
- kekurIa 作るもの。生成器
- sIf 白
- sifarE 白い
- sifarU yUsin 白色語
- yUsin 讀み書き。言語。言葉

```sh
./tasks.py help
./tasks.py clean build sh
./tasks.py start
```
